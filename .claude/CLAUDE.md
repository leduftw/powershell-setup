# powershell-setup

Personal PowerShell environment configuration for Windows.

## Repo contents

- `Microsoft.PowerShell_profile.ps1` — PowerShell profile, should be symlinked or copied to `$PROFILE`
- `ohmyposh.json` — Oh My Posh prompt theme, referenced by the profile using its full repo path

## Conventions

- Keep README.md up to date when adding or removing files
- The Oh My Posh config path in the profile points to the cloned repo location — update it if the repo is moved
